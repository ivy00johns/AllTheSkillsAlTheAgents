# Bold Direction Examples

The "leaned-in" direction — committed enough that it has a clear point of view, but still production-buildable. Use this slot when one of N directions needs to push the moat hard without going experimental.

## When the Bold Direction Lives Here

A "bold" direction means: a layout system or color discipline that announces what the project is about on the first scroll. Specifically:

- **Layout:** split 60/40 dossier, asymmetric broadsheet, or grid-bento — patterns that read as "deliberately not generic SaaS"
- **Typographic register:** monospace dossier, oversized italic broadsheet, or slab editorial — anything that signals position
- **Image treatment:** asymmetric photographic crops with bleed-off, or iconographic illustrations
- **Color discipline:** blackout + signal accent (red, sodium, signal-yellow) — high contrast, one chromatic note
- **Density:** file-document density or operator-tool dense — every pixel earning its keep

The bold direction's job is to be the version that makes the moat unignorable. If the safe direction is "this could be published in the magazine you know," the bold direction is "this could only be from THIS project."

## Worked Sample — {political-advocacy-site} "Blackout Dossier"

```text
Direction B: Blackout Dossier
Palette: near-black (#0A0A0A) + ivory (#F2EFE4) + signal red (#D62828)
Display: GT Sectra Bold (fallback: Source Serif 4 800)
Body: GT America Mono (fallback: JetBrains Mono 400)
Mono: same as body — register is unified
Layout: 60/40 split: photo on left full-bleed; right column is dossier text with monospace metadata strip
Hero: blacked-out photo with a single signal-red caption frame
Density: file-document — captions tight against photos, no decorative whitespace
Live elements: none on canvas — production version would have a live mortar-shell counter; mark "static still" in caption
CTAs: hard CTA bar at top "Support the front" with sticky behavior on production (static placement on canvas)
Photos: real photos, blacked-out treatment, caption frames in canvas
Tone reference apps: Kyiv Independent + The Counteroffensive Substack
```

## What Makes the Bold Direction Distinct

The variation rubric requires direction-to-direction differences on at least three orthogonal axes. The bold direction's "axis positions":

- **Layout axis:** split dossier (deliberately not editorial-column)
- **Register axis:** monospace (deliberately not serif gravity)
- **Treatment axis:** full-bleed with overlay frames (deliberately not half-bleed-with-caption)
- **Color axis:** blackout + signal (deliberately not cream + navy)

Every axis is positioned away from the safe direction. The bold direction is not "safe with more contrast" — it's a different position on every dimension. See `references/direction-examples/safe.md` for the contrast, and `references/direction-examples/experimental.md` for the third pole.
